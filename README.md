# Panda Doctor Expo App

This project is an Expo React Native application scaffolded in January 2026. It is ready for development and testing on Windows.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server for Android:
   ```sh
   npm run android
   ```
3. Open the app in your Android emulator or Expo Go app.

## Project Structure
- App.js: Main entry point
- package.json: Project configuration and dependencies

## Requirements
- Node.js (latest LTS recommended)
- npm
- Expo CLI
- Android Studio or Android emulator
